브라우저 안에서 바로 쓰는 OTP

Snap OTP는 Chrome 안에서 TOTP 코드를 저장하고 생성할 수 있는 가벼운 확장 프로그램입니다. 설정 페이지에서 QR 이미지를 업로드하거나 otpauth:// URI를 직접 붙여넣어 계정을 추가할 수 있고, 현재 탭의 QR 코드를 선택해서 바로 가져올 수도 있습니다.

저장된 항목은 팝업에서 바로 확인할 수 있고, 현재 코드를 복사하거나 항목 이름을 바꾸고 색상을 지정하고 순서를 정리할 수 있습니다. 설정 페이지에서는 보호 모드, 백업/복구, 전체 항목 관리까지 함께 처리할 수 있습니다.

기본 저장 방식

chrome.storage.sync를 사용해 항목을 저장하고 Chrome 프로필을 따라 동기화합니다.

선택형 보호 모드

원하면 패스프레이즈 보호를 켜서 저장된 OTP 항목을 암호화하고 팝업 잠금을 사용할 수 있습니다.

서버 없이 처리

QR 디코딩과 TOTP 생성은 모두 사용자 기기에서 처리되며, 개발자 서버로 OTP 시크릿을 보내지 않습니다.

주요 기능

Snap OTP는 빠른 일상 사용에 맞춘 단순한 흐름을 목표로 합니다. 계정을 추가하고, 코드를 확인하고, 필요한 순간에 바로 복사하는 작업을 Chrome 안에서 처리할 수 있습니다.

저장 및 보안

Snap OTP는 과장된 보안 약속 대신, 현재 구현 범위 안에서 저장 방식과 보안 경계를 명확히 설명합니다.

Standard mode

기본 모드에서는 OTP 항목이 chrome.storage.sync에 저장됩니다. 이 방식은 편의성과 동기화를 우선하며, Chrome 프로필과 Google 계정이 인증기 보안 경계의 일부가 됩니다.

Sync storage No extra passphrase
Passphrase protection

보호 모드를 켜면 저장된 OTP 항목이 sync 저장 전 암호화되고, 팝업은 패스프레이즈를 입력해야 다시 열립니다. 임시 잠금 해제 상태는 세션 저장소에만 유지되며 자동 잠금 시간이 지나면 제거됩니다.
